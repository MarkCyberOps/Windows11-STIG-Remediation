# Cyber Range: Windows 11 STIG Remediation

This repository contains PowerShell scripts developed to automate the remediation of Security Technical Implementation Guide (STIG) findings identified through Tenable vulnerability scans on virtual machines within the Cyber Range environment.

A **Security Technical Implementation Guide (STIG)** is a set of cybersecurity configuration standards published by the Defense Information Systems Agency (DISA). These guidelines are designed to ensure that systems, applications, and networks within the U.S. Department of Defense (DoD) are securely configured and maintained.

**Purpose of STIGs:**

  - Establish consistent and standardized security configurations across IT assets

  - Minimize exploitable vulnerabilities within systems and environments

  - Ensure compliance with DoD cybersecurity and regulatory requirements
